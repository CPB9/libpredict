Debian repo

We have an apt repo with binaries for amd64. To use it, please add the following to /etc/apt/sources.list.d/libpredict.list:

deb [arch=amd64] http://la1k.github.io/libpredict/debian sid main

The packages are signed by one of the developers. To install the key, run this command as root:

wget -O - http://ryeng.name/keys/CEB746D9.asc | apt-key add -
